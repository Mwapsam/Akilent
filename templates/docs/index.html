Akilent's Developer Platform lets you send transactional email three ways: a versioned REST API, direct SMTP relay, and signed webhooks for delivery, open, and click events. All three sit on top of the same domain verification and plan limits as the web dashboard.

Quick start

  1. Add and verify a sending domain on the Domains page.
  2. Generate an API key from the same page (see Authentication).
  3. Send your first message with POST /api/v1/messages.

What's not here yet

Client SDKs aren't published — every example on this site is a plain curl/HTTP request, and any language with an HTTP client and HMAC-SHA256 (for webhook verification) can integrate directly. WhatsApp isn't part of the public API yet — this platform currently covers email only.
