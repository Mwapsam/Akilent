Every error response from /api/v1/ uses the same envelope:

{
  "error": {
    "code": "unverified_domain",
    "message": "'yourdomain.com' is not a verified sending domain for this account"
  }
}
StatusCodeMeaning
400invalidRequest validation failed — check the message for which field.
401authentication_failedMissing or invalid API key.
403unverified_domainThe from address's domain isn't verified for your account.
403email_apisYour plan doesn't include the REST API & SMTP relay.
403emailsMonthly send volume cap reached — upgrade or wait for the next period.
404—Unknown API version (e.g. requesting /api/v2/... before it exists).
429—Per-key request-rate limit exceeded. Check Retry-After.

See Rate limits for the difference between the 429 (rate) and 403 emails (volume) cases — they're two separate limits.
